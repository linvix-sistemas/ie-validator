### Instalação:
```
yarn add @linvix-sistemas/ie-validator
```

### Uso
```
import IEValidator from '@linvix-sistemas/cpf-cnpj-validator';

// valida uma IE
const valid = IEValidator.validate('SC', '586653554');

```
